# ScenarioCraft

Kid-invented game scenarios for Paper (Minecraft-compatible) servers, judged by AI.

Our family invents the games; an AI coding agent builds them; GPT-5.6 referees the matches.
First scenario: an AI-judged **Build Battle**. More kid-invented scenarios will live here over time.

This repository is being built in the open for **OpenAI Build Week (July 2026)** — the code
arrives via Codex, starting from issue #1.
