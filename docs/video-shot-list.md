# ScenarioCraft Build Week video shot list

Target runtime: **2:45**. Record in 16:9 with spoken narration throughout. Use invented player
names only; show no private server address, API key, child name, location, or photo of a person.

| Time | Picture | Required narration |
| --- | --- | --- |
| 0:00–0:15 | Open the Speed Build How to Play hero, then scroll into the seven-step story. | “Our kids invent the game. Codex builds it. GPT-5.6 referees it. ScenarioCraft turns that loop into a family game night.” |
| 0:15–0:45 | Start a real round at the Secret Chest; show the chosen player, prompt reveal, and private plots. | Explain that Speed Build is the first scenario, with chat/title/bossbar controls that work for Java and Bedrock players. |
| 0:45–1:15 | Place blocks, let the timer finish, and show wall removal plus the gallery tour. | Explain the kid-authored rules: one secret prompt, no peeking, admire every build before scoring. |
| 1:15–1:45 | Show the committed voxel export, renderer output, and three judge verdicts; return to the in-game winner announcement. | “The running judge CLI sends every build to three personas on GPT-5.6. They share one rubric, require a quorum, and every comment names a real strength.” |
| 1:45–2:15 | Open the GitHub issue list, one delivery PR with its session receipt, an ExecPlan Decision Log, and `AGENTS.md` Corrections. | “Codex built this clean-room repository from issues. It accelerated the work by delivering tested vertical slices, while the issue-to-PR trail, review policy, and session IDs kept every decision inspectable.” |
| 2:15–2:30 | Show `docs/plans/pages-how-to-play.md` at the local-preview discovery and the fixed self-contained `site/` tree. | Explain the discipline-caught win: review found that the logo worked only during deployment, so Codex fixed local preview before merge and recorded the failure. |
| 2:30–2:45 | Land on “The crafting recipe,” open the Codex and GPT-5.6 ingredient cards, then frame the Speed Build output slot. | “One family’s game night: specified by kids, built by a coding agent, refereed by AI. That is the recipe we want other families to remix.” |

## Capture checklist

- [ ] Working round is visible; do not rely on slides alone.
- [ ] Voiceover says what the project does, how Codex was used, and how GPT-5.6 is integrated.
- [ ] Codex workflow evidence includes an issue, PR, session receipt, and one caught problem.
- [ ] GPT-5.6 evidence includes the configured model, three personas, shared rubric, and verdict.
- [ ] Video is under three minutes and uploaded publicly to YouTube.
- [ ] Public YouTube URL is added to `README.md`, `SUBMISSION.md`, and the Devpost form.
- [ ] Minecraft disclaimer is visible or spoken; no unlicensed music or third-party footage is used.
