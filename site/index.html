Ready, set, build

A Speed Build round in 60 seconds

No game download needed for this tour. Every picture below was drawn for ScenarioCraft or rendered from its own voxel data.

Build Week became our build weekend: first commit Saturday afternoon, a playable round by Sunday night, submitted Tuesday. The last two days made it provable—with judges, renderer views, and a demo a stranger can run.

One household, every device

Everybody builds together

Different kids can join the same kid-safe server from iPad, Windows Bedrock, or a Java PC, and it still feels like a real online game. The secret prompt, timers, and warm AI panel judge every build the same way, whatever device its builder uses. Consoles need an operator-supplied LAN discovery or redirect setup.

    Everybody in

    Everyone meets at the Secret Chest

    /speedbuild start brings every player to the arena. One chest waits in the middle, hiding tonight's secret challenge.

  2. 02

    The lucky click

    One player is chosen to open it

    The real transcript chose Pixel. Blocky tried first and the gate answered This secret note belongs to Pixel this round. You'll see the idea together soon!

    Pixel is the secret-note picker! The chest is waiting at the hub.

  3. 03

    The big reveal

    The secret challenge is revealed

    For this walkthrough, imagine the title filling every screen with one colorful prompt: A rainbow volcano.

  4. 04

    No peeking

    Everyone builds behind giant walls

    Each builder gets a private walled plot with an anti-peek roof. A bossbar keeps the countdown visible while blocks turn into ideas.

    BUILD! 1:29
  5. 05

    Psst, camera crew

    Camera cutaway

    We can slice through the block-by-block voxel export to show a build in progress. Players can never see this view during the round. Nice try, peekers.

  6. 06

    Tour time

    Time's up. The walls disappear

    Block edits stop, the dividers dissolve, and everybody tours the gallery. This is the admire-first part: every build made it from a blank plot.

  7. 07

    Warm, fair, funny

    Three AI judges score every build

    The GPT-5.6 panel uses one shared 1–10 rubric: theme fit, creativity, effort, and detail. The winner gets a celebration. Every builder gets one genuine strength and one useful tip.

Your family is next

Run it at home tonight

These are the same three steps as the repository quickstart.

  1. Clone git clone https://github.com/agorokh/scenariocraft.git && cd scenariocraft
  2. Start the local server export OPENAI_API_KEY='<your OpenAI API key>' docker compose up --build
  3. Play Join localhost:25565 in Minecraft Java 1.21.x. Run /speedbuild start in chat. /battle and /bb remain available for existing servers.
