# Shared judging rubric

Use exactly the same four criteria for every persona and every build. Give each criterion an
integer score from 1 to 10. Persona direction changes only the voice of the commentary, never
what earns points or how strictly the build is scored.

## theme_fit

How clearly the build answers the task.

- **3:** The task is hard to recognize, or only one small part connects to it.
- **6:** The main idea matches the task, with some parts that feel unrelated or unfinished.
- **9:** The task is immediately clear, and nearly every part helps bring it to life.

## creativity

How inventively the builder develops the task with their own choices.

- **3:** The build uses a very familiar idea with few personal or surprising choices.
- **6:** The build includes some original choices that make the idea feel personal.
- **9:** The build combines bold, surprising choices into an idea that feels unmistakably its own.

## effort

How much care and sustained work is visible in the completed build.

- **3:** The build has only a few placed elements and looks mostly unfinished.
- **6:** The build shows steady work across its main parts, though some areas remain unfinished.
- **9:** The build shows exceptional care across the whole scene, including ambitious or time-consuming parts.

## detail

How thoughtfully smaller features, materials, and spaces support the whole build.

- **3:** The build has few smaller features, with large plain or empty areas.
- **6:** The build has several purposeful features or material choices, with room for more polish.
- **9:** The build is rich in purposeful features, varied materials, and polished spaces that reward a closer look.

## Kid-facing tone and verdict order

Reason before scoring: first explain what is visible in the build and why it supports the evaluation,
then assign numbers. Every comment must name at least one genuine, concrete strength. Never mock a
builder, never use sarcasm about effort, and never guilt, shame, or compare one child with another.
When suggesting an improvement, offer one kind, understandable next step rather than a list of faults.
